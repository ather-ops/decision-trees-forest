Decision trees and random forest implementations from scratch and with scikit-learn.

## Learning Path

**Week 1: Decision Tree**
- Day 1: Tree structure, impurity measures
- Day 2: Conditions (axis-aligned vs oblique, binary vs non-binary)
- Day 3: Growing tree, greedy divide-and-conquer
- Day 4: Training algorithm, recursive build
- Day 5: Exact splitter, entropy, information gain
- Day 6: Overfitting, pruning, regularization
- Day 7: Feature importance

**Week 2: Random Forest**
- Day 8: Ensemble learning, bagging
- Day 9: Bootstrap sampling
- Day 10: Feature randomization
- Day 11: Aggregation (majority voting, averaging)
- Day 12: Out-of-bag evaluation
- Day 13: Hyperparameter tuning
- Day 14: Complete project

## Project: Drug Classifier

Predicts drug type (drugY, drugX, drugC, drugA, drugB) from patient data:
- Age, Sex, BP, Cholesterol, Na_to_K ratio

**Accuracy:** ~70% with simple decision tree
